# Eco-Forensics Dashboard

A React-based dashboard for visualizing environmental forensic data, linking deforestation and fire events to corporate entities.

## Features

- **Interactive Map**: Visualize fires, deforestation alerts, and infrastructure on a Leaflet map
- **Evidence Chains**: See connections between industrial facilities and environmental damage
- **Sentinel Analysis**: View satellite imagery indices (NDVI, NBR, Burn Index)
- **Impact Charts**: Analyze trends in forest loss and fire activity
- **Sentiment Tracking**: Monitor media coverage sentiment
- **Data Source Status**: Real-time health monitoring of all data sources

## Prerequisites

- Node.js 18+ 
- npm or yarn
- Backend API running (see backend README)
